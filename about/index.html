Arpan Dhatt / About

I'm Arpan Dhatt, an incoming freshman to UT Austin Computer Science. This website is mostly just so I can write random stuff that comes to mind, which usually happens to be CS-focused or at least CS-related. I spent a lot of time (read: too much time) making micro-optimizations to this site to make it easier on the eyes, just for readers like you! If you have suggestions, let me know! I'd love to hear them.

But anyway, I've been interested in computers for almost half my life at this point, and over the years, I've learned a few things and worked on a few projects. I tried game programming for a little while (using Unity mostly), and decided it wasn't for me, although making a space-shooter with missiles and stuff was really cool. For a few years now, I've been learning and occasionally applying machine learning to various projects. Most of the recent projects involving ML/DL have been hackathons. Check out my Github homepage for a few pinned repo's with projects (quite a few wins, too) in them.
